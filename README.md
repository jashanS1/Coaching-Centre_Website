## ğŸš€ Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Safari, Edge)
- A text editor (optional, for customization)

### Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Open `index.html` in your web browser

### Local Development

Simply open the HTML files in your browser. No build process or server required.

For better development experience, you can use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## ğŸ¨ Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --royal-blue: #004aad;
    --white: #ffffff;
    --golden-yellow: #f5c518;
    /* ... other colors ... */
}
```

### Adding Images

Replace the placeholder icons and backgrounds with actual images:

1. Add your images to a folder (e.g., `images/`)
2. Update the image paths in the HTML files
3. Modify the `.gallery-image`, `.course-image`, etc. classes to use background images

### Modifying Content

Edit the HTML files directly to update:
- Text content
- Contact information
- Course details
- Team member information
